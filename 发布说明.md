## 📤 分发给用户

### 上传到 GitHub Release

1. **创建 Release**
```bash
git tag v1.0.0
git push origin v1.0.0
```

2. **上传 DMG**
- 前往 GitHub Releases 页面
- 点击 "Create a new release"
- 选择 v1.0.0 标签
- 上传 `Sidekick-1.0.0-macOS.dmg`
- 添加发布说明

### 用户安装步骤

```
1. 下载 Sidekick-1.0.0-macOS.dmg
2. 双击打开 DMG 文件
3. 将 Sidekick 拖拽到 Applications 文件夹
4. 在 Launchpad 或 Applications 中找到 Sidekick
5. 双击运行（首次可能需要在系统设置中允许）
```

### 首次运行提示

macOS 可能会显示 "无法验证开发者" 的警告：

**解决方法：**
1. 前往 系统设置 → 隐私与安全性
2. 找到 "仍要打开" 按钮
3. 点击打开

或者使用命令行：
```bash
xattr -cr /Applications/Sidekick.app
```
