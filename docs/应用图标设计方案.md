# Sidekick 应用图标设计方案

## 🎨 设计理念

**核心概念**: 助手、伙伴、可靠、专业

**视觉元素**: 
- 主图标：盾牌 + 人物剪影（代表保护和协助）
- 配色：渐变蓝色（专业、可信赖）
- 风格：现代、简洁、macOS 原生

---

## 📐 图标规格

### macOS 应用图标尺寸
- 1024x1024 (App Store)
- 512x512 @2x
- 256x256 @2x
- 128x128 @2x
- 32x32 @2x
- 16x16 @2x

### 设计要求
- 圆角矩形（macOS 标准）
- 支持浅色和深色模式
- 清晰的轮廓，易于识别
- 在小尺寸下仍然清晰

---

## 🎯 设计方案

### 方案 A: 盾牌 + 人物（推荐）⭐⭐⭐⭐⭐

**视觉描述**:
```
┌─────────────────────┐
│                     │
│      🛡️ + 👤        │
│                     │
│   渐变蓝色背景       │
│   白色图标元素       │
│                     │
└─────────────────────┘
```

**配色方案**:
- 背景：线性渐变
  - 起始色：#4A90E2 (明亮蓝)
  - 结束色：#2E5C8A (深蓝)
- 图标：白色 #FFFFFF
- 阴影：半透明黑色，增加立体感

**设计元素**:
1. **盾牌**: 占据图标中心，代表保护和可靠
2. **人物剪影**: 在盾牌内部或旁边，代表助手
3. **对勾**: 可选，在盾牌上添加对勾，代表完成任务

**优点**:
- 直观传达"助手"概念
- 专业且现代
- 符合 macOS 设计语言
- 在小尺寸下仍然清晰

### 方案 B: 字母 SK + 工具图标 ⭐⭐⭐⭐

**视觉描述**:
```
┌─────────────────────┐
│                     │
│        SK           │
│      🔧 🔨          │
│                     │
│   渐变背景           │
│                     │
└─────────────────────┘
```

**配色方案**:
- 背景：渐变蓝色
- 字母 SK：白色，粗体
- 工具图标：浅色，半透明

**设计元素**:
1. **字母 SK**: 大号字体，占据主要位置
2. **工具图标**: 扳手、锤子等，暗示工具箱概念
3. **圆角矩形**: macOS 标准样式

**优点**:
- 品牌识别度高
- 简洁明了
- 易于记忆

**视觉描述**:
```
┌─────────────────────┐
│                     │
│      ◆ ◇ ◆          │
│     ◇ ◆ ◇           │
│      ◆ ◇            │
│                     │
└─────────────────────┘
```

**配色方案**:
- 背景：深蓝色
- 几何图形：多色渐变

**设计元素**:
1. **几何图形**: 菱形、六边形组合
2. **渐变色**: 蓝色到紫色
3. **动态感**: 图形排列暗示流动和协作

**优点**:
- 现代感强
- 独特性高
- 适合科技产品

## 📦 图标资源包结构

创建图标后，需要准备以下文件：

```
Sidekick.iconset/
├── icon_16x16.png
├── icon_16x16@2x.png
├── icon_32x32.png
├── icon_32x32@2x.png
├── icon_128x128.png
├── icon_128x128@2x.png
├── icon_256x256.png
├── icon_256x256@2x.png
├── icon_512x512.png
├── icon_512x512@2x.png
└── icon_1024x1024.png
```

### 生成 .icns 文件

```bash
# 将 iconset 转换为 .icns 文件
iconutil -c icns Sidekick.iconset
```

## ✅ 设计检查清单

在最终确定图标前，检查以下项目：

- [ ] 在不同尺寸下都清晰可辨
- [ ] 在浅色和深色背景下都好看
- [ ] 符合 macOS 设计规范
- [ ] 与品牌定位一致
- [ ] 独特且易于识别
- [ ] 在 Dock 中显示效果好
- [ ] 在 Finder 中显示效果好
- [ ] 在 App Store 中显示效果好
