## 📅 Week 2：SQL 查询引擎

### 任务清单

- [ ] 集成 SQLite
- [ ] 实现数据加载到内存数据库
- [ ] 实现 SQL 查询执行
- [ ] 实现查询结果展示

### 技术要点

**SQLite 集成**：
```swift
import SQLite3

var db: OpaquePointer?
sqlite3_open(":memory:", &db)
```

**创建表**：
```swift
let createTableSQL = """
CREATE TABLE users (
    id INTEGER,
    name TEXT,
    age INTEGER
)
"""
sqlite3_exec(db, createTableSQL, nil, nil, nil)
```
