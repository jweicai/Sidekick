# Sidekick UI 设计提示词

## 项目概述
设计一个现代化的 macOS 开发者工具应用 UI，名为 "Sidekick"。这是一个多功能开发助手，集成了数据查询、JSON/IP/时间戳处理等常用开发工具。

## 核心功能需求

### 主要功能模块
1. **文件管理区域** - 拖拽上传、文件列表、表结构预览
2. **SQL 查询编辑器** - 代码编辑、语法高亮、查询历史
3. **结果展示区域** - 数据表格、分页、排序、筛选
4. **导出工具栏** - CSV/JSON/SQL 导出、格式选择
5. **多表管理面板** - 表关系图、JOIN 辅助

### 用户工作流程
1. 拖拽或选择 CSV/JSON 文件 → 2. 查看表结构和数据预览 → 3. 编写 SQL 查询 → 4. 执行查询查看结果 → 5. 导出数据

## UI 设计要求

### 整体风格
- **设计语言**: 现代简约，遵循 macOS Human Interface Guidelines
- **色彩方案**: 以深色/浅色主题为主，支持系统主题切换
- **主色调**: 专业蓝色 (#007AFF) 作为强调色，灰色系作为主体
- **字体**: SF Pro Display/SF Mono（代码区域）
- **圆角**: 8px-12px 统一圆角设计
- **阴影**: 微妙的投影效果，营造层次感

### 详细组件设计

#### 1. 顶部工具栏 (44px 高度)
- **左侧**: Sidekick Logo + 应用名称
- **中间**: 面包屑导航（当前工作空间/项目）
- **右侧**: 
  - 添加文件按钮 (⌘+N)
  - 主题切换开关
  - 设置齿轮图标
  - 帮助问号图标

#### 2. 左侧边栏 (280px 宽度，可调节)
- **已加载表区域**:
  - 表名 + 行数/列数统计
  - 展开/折叠按钮查看列详情
  - 每列显示：列名、数据类型、示例值
  - 右键菜单：重命名、删除、导出
- **查询历史区域**:
  - 时间戳 + 查询预览
  - 收藏/取消收藏功能
  - 搜索历史查询
- **空状态**:
  - 拖拽区域提示
  - 支持格式说明
  - 示例文件链接

#### 3. SQL 查询编辑器
- **编辑器特性**:
  - 语法高亮（SQL 关键字、表名、列名）
  - 自动补全（表名、列名、SQL 函数）
  - 行号显示
  - 代码折叠
  - 多标签页支持
- **工具栏**:
  - 执行按钮 (▶️) + 快捷键提示
  - 格式化代码按钮
  - 保存查询按钮
  - 查询模板下拉菜单
- **状态指示**:
  - 执行中的加载动画
  - 语法错误红色下划线
  - 成功/失败状态图标

#### 4. 查询结果区域
- **结果表格**:
  - 虚拟滚动支持大数据集
  - 列宽自适应 + 手动调节
  - 排序箭头指示器
  - 行斑马纹效果
  - NULL 值特殊显示（斜体灰色）
- **分页控件**:
  - 页码选择器
  - 每页行数选择
  - 总数统计显示
- **筛选工具**:
  - 列头筛选下拉菜单
  - 快速搜索框
  - 数据类型筛选

#### 5. 底部状态栏 (32px 高度)
- **左侧信息**:
  - 查询执行时间
  - 结果行数统计
  - 数据库连接状态
- **右侧操作**:
  - 导出按钮组：CSV | JSON | SQL
  - 全屏切换按钮
  - 缩放控制

### 交互设计细节

#### 文件拖拽体验
- **拖拽悬停**: 整个窗口高亮边框，显示"释放以加载文件"
- **加载动画**: 进度条 + 文件解析状态
- **错误处理**: Toast 通知 + 详细错误信息面板

#### 查询执行流程
- **执行前**: 按钮变为加载状态，禁用编辑器
- **执行中**: 进度指示器 + 可取消按钮
- **执行后**: 结果淡入动画 + 成功提示音

#### 响应式设计
- **窗口最小尺寸**: 1200x800px
- **侧边栏**: 可折叠，最小 60px（仅图标）
- **编辑器/结果**: 可调节分割比例
- **全屏模式**: 隐藏工具栏，最大化工作区域

### 视觉层次

#### 色彩系统
```
主色调:
- Primary Blue: #007AFF (按钮、链接、强调)
- Success Green: #34C759 (成功状态)
- Warning Orange: #FF9500 (警告)
- Error Red: #FF3B30 (错误)

中性色:
- Text Primary: #000000 (浅色模式) / #FFFFFF (深色模式)
- Text Secondary: #6C6C70 / #98989D
- Background: #FFFFFF / #1C1C1E
- Surface: #F2F2F7 / #2C2C2E
- Border: #C6C6C8 / #38383A
```

#### 字体层级
```
- 标题 1: SF Pro Display Bold 28px
- 标题 2: SF Pro Display Semibold 22px
- 标题 3: SF Pro Display Medium 18px
- 正文: SF Pro Text Regular 14px
- 说明文字: SF Pro Text Regular 12px
- 代码: SF Mono Regular 13px
```

### 动效设计
- **页面切换**: 0.3s ease-in-out 淡入淡出
- **按钮交互**: 0.15s ease-out 缩放反馈
- **数据加载**: 骨架屏 + 渐进式加载
- **错误提示**: 0.2s 弹性动画

### 可访问性
- **键盘导航**: 完整的 Tab 键导航支持
- **屏幕阅读器**: 语义化标签和 ARIA 属性
- **对比度**: 符合 WCAG 2.1 AA 标准
- **字体缩放**: 支持系统字体大小设置

### 特色功能 UI

#### 智能查询助手
- **浮动面板**: 右下角智能助手图标
- **查询建议**: 基于表结构的 SQL 模板
- **错误诊断**: 语法错误的智能修复建议

#### 数据可视化预览
- **迷你图表**: 数值列的分布直方图
- **数据质量**: 空值率、唯一值统计
- **关系图**: 多表间的关联关系可视化

这个设计提示词涵盖了专业数据工具的所有关键 UI 元素，既保持了 macOS 原生应用的体验，又突出了数据查询工具的专业性和易用性。
