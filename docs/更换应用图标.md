# 更换应用图标指南

## 步骤

### 1. 保存图标文件

将你的图标图片保存到项目根目录，命名为 `app-icon.png`

**要求：**
- 格式：PNG
- 尺寸：建议 1024x1024 或更大
- 背景：透明或纯色

### 2. 运行生成脚本

```bash
./generate-icons.sh
```

脚本会自动生成所有需要的尺寸：
- 16x16, 32x32, 64x64, 128x128, 256x256, 512x512, 1024x1024

### 3. 重新编译

```bash
swift build
```

或在 Xcode 中重新构建（⌘+B）

### 4. 验证

运行应用后，在 Dock 和应用切换器中应该能看到新图标。

---

## 手动操作（如果脚本不工作）

### 使用在线工具

1. 访问 https://appicon.co/
2. 上传你的图标
3. 选择 "macOS"
4. 下载生成的文件
5. 解压并复制到 `Sources/Resources/Assets.xcassets/AppIcon.appiconset/`

### 使用 macOS 预览

1. 打开你的图标文件
2. 工具 → 调整大小
3. 手动创建每个尺寸并保存

## 故障排查

### 图标没有更新？

1. **清理构建缓存**
   ```bash
   rm -rf .build
   swift build
   ```

2. **清理 Xcode 缓存**
   ```bash
   rm -rf ~/Library/Developer/Xcode/DerivedData/Sidekick-*
   ```

3. **重启 Dock**
   ```bash
   killall Dock
   ```

### sips 命令不存在？

`sips` 是 macOS 自带的图像处理工具，应该默认可用。如果不行，使用在线工具。
