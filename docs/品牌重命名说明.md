# Sidekick 品牌重命名说明

## 📋 重命名概述

**原名称**: TableQuery  
**新名称**: Sidekick  
**重命名日期**: 2025-01-14

---

## 🎯 重命名原因

### 1. 产品定位变化
- **原定位**: 数据表查询工具
- **新定位**: 开发者综合工具箱

### 2. 功能范围扩展

**核心功能（~40%）**:
- 数据表加载（CSV、JSON、XLSX）
- SQL 查询
- 多表 JOIN
- 数据导出

**开发工具（~60%）**:
- JSON 工具（扁平化、格式化、压缩、验证、路径查询）
- IP 工具（格式转换、子网计算、地址验证、批量处理）
- 时间戳工具（时间戳转换、时区处理）
- 文本工具（Base64、URL编码、Hash、文本对比）
- 其他工具（UUID、颜色转换、正则测试）

### 3. 品牌识别度
- "TableQuery" 只体现了表格查询功能，容易被误解为专业数据库工具
- "Sidekick" 准确传达了"开发助手"的定位
- 更容易建立情感连接和品牌认同

---

## 🎨 新品牌定位

### 品牌名称
**Sidekick** - 你的开发助手

### 品牌 Slogan
- **英文**: *"Sidekick - Your Coding Companion"*
- **中文**: *"Sidekick - 你的编码伙伴"*

### 品牌故事
> 每个超级英雄都有一个可靠的助手。在开发的世界里，Sidekick 就是你的得力伙伴，随时准备帮你处理那些繁琐但必要的任务。

### 产品描述
> Sidekick 是一款为 macOS 开发者打造的多功能工具箱，集成了数据查询、格式转换、编码工具等常用功能。无论是 SQL 查询、JSON 处理、时间戳转换还是 IP 地址计算，Sidekick 都能帮你快速完成。
