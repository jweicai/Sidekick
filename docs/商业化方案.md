## 许可证系统

### 技术方案
- **本地验证**：不需要服务器，离线可用
- **激活码格式**：XXXX-XXXX-XXXX-XXXX
- **加密方式**：HMAC-SHA256 签名
- **存储方式**：UserDefaults（本地加密存储）

### 激活流程
1. 用户购买后收到激活码
2. 在应用设置中点击"激活"
3. 输入激活码
4. 本地验证通过后激活成功
5. 许可证信息保存到本地

### 防盗版措施
- 激活码签名验证
- 一码一用（可选：绑定设备）
- 定期检查许可证有效性
- 混淆关键代码

## 销售渠道

### 方案 A：独立销售（推荐）
**平台**：Gumroad / Paddle / LemonSqueezy  
**优点**：
- 手续费低（5-10%）
- 完全控制定价和促销
- 可以收集用户邮箱
- 支持多种支付方式

**缺点**：
- 需要自己处理客服
- 需要自己做营销

### 方案 B：Mac App Store
**优点**：
- 苹果官方渠道，信任度高
- 自动更新
- 全球分发

**缺点**：
- 手续费高（30%）
- 审核严格
- 定价受限

### 推荐策略
1. **初期**：独立销售（Gumroad）
2. **成熟期**：同时上架 Mac App Store
3. **企业版**：直接销售 + 定制服务
