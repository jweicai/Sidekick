# 如何查看 Sidekick 应用图标

## 方法一：使用 Xcode 运行（推荐）⭐⭐⭐⭐⭐

**Xcode 已自动打开！**

### 步骤：
1. ⏳ **等待 Xcode 加载** - 等待左上角的进度条完成
2. 🎯 **选择 Scheme** - 确认顶部工具栏选中 "Sidekick"  
3. ▶️ **运行应用** - 点击运行按钮或按 `⌘+R`
4. 🎉 **查看图标** - 应用启动后，在 Dock 中就能看到蓝色盾牌图标了！

### 如果图标还是显示为 "exec"：
```
Product -> Clean Build Folder (⌘+Shift+K)
然后重新运行 (⌘+R)
```

---

## 方法二：创建 .app 应用包

如果你想要一个独立的 .app 文件：

### 使用 Xcode 导出：
1. 在 Xcode 中选择 `Product` -> `Archive`
2. 等待归档完成
3. 点击 `Distribute App`
4. 选择 `Copy App`
5. 选择保存位置

### 或者使用脚本（需要修复）：
```bash
./build-app.sh
```

## 常见问题

### Q: 为什么 `swift run` 看不到图标？
A: `swift run` 创建的是命令行可执行文件，不包含图标。需要通过 Xcode 运行或创建 .app 包。

### Q: 如何更换图标？
A: 
1. 使用图标生成工具（如 https://appicon.co/）
2. 替换 `Sources/Resources/Assets.xcassets/AppIcon.appiconset/` 中的 PNG 文件
3. 在 Xcode 中重新构建

### Q: 图标模糊或不清晰？
A: 确保提供了所有尺寸的图标，特别是 @2x 版本（Retina 显示器需要）
